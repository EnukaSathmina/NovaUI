Quick Start Guide

Get NovaUI up and running in less than a minute.

1 Include Stylesheets

Add the core CSS and the components you need to your HTML <head>.


<!-- Core Styles -->
<link rel="stylesheet" href="css/main.css">

<!-- Components -->
<link rel="stylesheet" href="components/buttons.css">
<link rel="stylesheet" href="components/cards.css">

2 Recommended Directory Structure

Keep your project organized for smooth asset linking.

  • 📁 project-root/
  • 📁 css/ (main.css)
  • 📁 components/ (buttons.css, cards.css...)
  • 📁 js/ (animations.js)
  • 📄 index.html

3 Add Your First Component

Paste this into your body to test if everything is working.


<button class="btn btn-primary">
  NovaUI is Active!
</button>
