## 💎 Why NovaUI?

NovaUI isn't just another CSS library. It’s a design system focused on depth, light, and motion. By utilizing advanced `backdrop-filter` techniques, we provide a "frosted glass" look that works seamlessly across all modern browsers.

### 🚀 Core Strengths
- **Glassmorphism at Scale:** Perfectly tuned blur and transparency ratios.
- **Zero-Config Installation:** No NPM, no build tools, just pure CSS.
- **Developer First:** Documentation includes one-click "Copy Code" logic.
- **Mobile Optimized:** Fully responsive grid systems and navigation.

## 🏁 Getting Started

### 1. Project Setup
Organize your folders to ensure the paths match the library's internal logic:

```text
📁 novaui/
   ├── 📁 components/ (buttons.css, cards.css, etc.)
   ├── 📁 demo/ (button.html, card.html, navbar.html, etc.)
   ├── 📁 js/ (animations.js, dropdown.js, modal.js)
   └── 📁 styles/ (main.css)
```

### 2. Implementation
Link the core files in your ```<head>```:

```
<link rel="stylesheet" href="novaui/styles/main.css">
<link rel="stylesheet" href="novaui/components/buttons.css">
```
